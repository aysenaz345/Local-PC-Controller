# Local-PC-Controller
A Python-based desktop application for controlling mouse, keyboard, volume, and monitoring the screen, with future support for mobile remote control.

#  Remote PC Controller

## About

Remote PC Controller is a desktop application developed with Python that allows users to control basic computer functions through a simple graphical interface.

The application provides quick access to mouse controls, keyboard input, volume management, and screen monitoring. It was created as a learning project to explore desktop application development and computer automation using Python.

The current version is designed to work locally on the same computer. Future versions are planned to support controlling the computer remotely from a mobile device over a local Wi-Fi network.

---

## Features

-  Mouse movement controls
-  Mouse click support
-  Virtual keyboard buttons
-  Backspace and Space keys
-  Volume up, down, and mute controls
-  Live desktop preview (updated every few seconds)
-  Simple and user-friendly interface

---

## Technologies Used

- Python
- Tkinter
- PyAutoGUI
- Pillow (PIL)
- Socket (planned for future remote communication)

---

## Installation

Clone the repository:

```bash
git clone https://github.com/your-username/Remote-PC-Controller.git
```

Install the required libraries:

```bash
pip install pyautogui pillow
```

Run the application:

```bash
python main.py
```
